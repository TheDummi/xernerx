    • Client

    Client is an extension of the Discord Client.

    Parameter Type Default Required Description
    Intents Number[] None true Array of an intent number or Discord FLAGS.
    GuildId String None true Whether the bot should deploy it's Interaction commands globally or locally.
    Global Boolean None false Whether the bot should deploy it's Interaction commands globally or locally.
    Prefix String[] None false An array of prefixes that the bot should listen to.
    OwnerId String[] None false Array Owner Id's.
    IgnoreOwner Boolean false false Whether the bot should ignore all preconditions on owners.
    DefaultCooldown Number 0 false How long it takes before a user can run the command again. (in ms)
    Logging Boolean false false Whether the bot should log loading of items (enables globally)
    CacheTime Number 300000 false How long it takes before messages become invalid to edit. (in ms)
    Messages Object Object false An array of cache messages and their command message ID (cannot be set)
    Color Object FLAGS false Object of colors you want to use in your bot.
    Config Config Object false Any additional info you want to save to the bot.
